#  Ma Bibliothèque de Code Roblox (Luau)

Une collection de scripts Roblox réutilisables, écrits en **Luau** et entièrement commentés en français. Idéale pour démarrer un jeu de parcours (obby) ou un système de progression.

 >Chaque script indique en commentaire **où le placer** et **comment l'utiliser**.


 ---


 ##  Comment utiliser un script

1. Ouvre **Roblox Studio**.
2. Dans l'**Explorer**, crée un **Script** au bon emplacement (voir le tableau).
3. Copie-colle le contenu du fichier `.lua` correspondant.
4. Lis les commentaires en haut du fichier : ils expliquent la mise en place.



##  Conseil sécurité

N'insère pas de modèles inconnus depuis la Toolbox sans vérifier qu'ils ne contiennent pas de scripts cachés.
une arnaque circule en ce moment ou on nous affiche un message d'érreur a l'écran demandant de coller du code dans la barre de commande -> il s'uffit de suprimer l'item quon a pris de la toolbox pour ne plus avoir cette affichage.
